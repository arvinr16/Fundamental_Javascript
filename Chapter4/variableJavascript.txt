>> Variabel-variabel dalam Javascript <<

=> Introduce
   Kita disini akan belajar mengenai variabel. 
   Apa itu variabel? Variabel adalah tempat untuk menampung sebuah data.
   Disaat kita mendeklarasikan sebuah variabel, variabel tersebut akan tersimpan di meory didalam browser-
   atau di engine Javascriptnya, variabel tersebut akan tersimpan didalam memory tersebut yang nantinya-
   kita bisa panggil untuk kita gunakan.
<======================================================================================================>


=> Subject Matter
   1. Kita bisa membuat file 'index.html', lalu kita bisa langsung menambahkan tag <script> pada file-
      tersebut.
   2. Materi lanjut ke file 'main.js' Chapter 4
   3. Didalam Javascript ada yang namanya 'Scoping', Scoping adalah aturan yang menentukan di mana-
      suatu variabel bisa diakses dalam suatu kode program. Biasanya Scoping dimulai dan diakhiri-
      dengan tanda kurung kurawal ({}).
      Ada dua jenis Scoping yaitu-
      1. Global Scoping = Variabel dideklarasikan di luar fungsi atau blok manapun dan dapat diakses-
                          dari mana saja di dalam suatu kode.
      2. Local Scoping  = Variabel dideklarasikan di dalam fungsi atau blok tertentu, jadi hanya bisa-
                          diakses didalam area tersebut saja.
