<== Chapter 3 ==>
>> Cara menjalankan Javascript <<

=> Introduce
   Untuk kita bisa menjalankan Javascript, sebelumnya kita harus mempunyai aplikasi Browser dan Text-
   Editor terlebih dahulu.
   Sebenarnya pada aplikasi Browser kita itu sudah terpasang Javascript atau Engine Javascript untuk-
   menjalankan kode Javascript yang ada di browser secara default.
   Ada 4 kategori fitur Javascript yang kita bisa jalankan pada Browser kita:
   1. Core (ECMAScript) = Fitur Javascript yang didefinisikan oleh standar ECMAScript.
   2. DOM (Document Object Model) = Digunakan untuk memanipulasi elemen halaman Web.
   3. BOM (Browser Object Moder)  = Digunakan untuk memberi Javascript akses ke fitur-fitur browser, 
                                    bukan hanya halaman webnya.
   4. Web API = Fitur-fitur tambahan browser agar Javascript bisa melakukan hal-hal canggih.
<======================================================================================================>


=> Subject Matter
   Kita bisa menjalankan Javascript pada Browser kita dengan cara:
   1. Masuk pada Browser kita
   2. Setelah itu pada landing page kita bisa membuka menu 'Inspect' dengan cara klik kanan lalu pilih-
      'Inspect'
   3. Pada menu Inspect kita bisa mengklik menu 'Console'
   4. Setelah pada menu Console tersebut kita bisa langsung membuat kode Javascript dan bisa langsung-
      tereksekusi oleh Browser kita.
   5. Tetapi pada saat kita merefresh Browser kita, kode yang kita buat tadi akan langsung ter reset,-
      sehingga kita tidak bisa memanggil variable yang sudah kita buat tadi di awal atau kita harus-
      membuat ulang variable tersebut.
   6. Kita juga bisa membuat sebuah fitur Javascript yang ada langsung menggunakan Console di browser-
      dengan cara kita menuliskan kode syntax 'alert('value')' seperti itu.
   7. Ketika kita ingin menggunakan Inline Javascript style code, misal kita ingin menggunakan 'console-
      .log('Hello'), kita jangan lupa bahwasanya kalau untuk attribut tag Html seperti onclick="" dia me-
      merlukan tanda kutip (""). Maka dari itu kita harus menulis dengan kebalikan tanda kutip ter-
      sebut dengan kode Javascript Inline kita, kalau tidak begitu kodenya akan error.
      Contoh yang benar: <a href="" onclick="console.log('Hello')"><a/>
      Contoh yang salah: <a href="" onclick="console.log("Hello")"><a/>
