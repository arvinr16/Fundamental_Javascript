>> Tipe Data dalam Javascript <<

=> Introduce
   Tipe data adalah bentuk Data yang disimpan dalam suatu variabel.
   Ada beberapa tipe data yang ada pada bahasa permograman Javascript.
   Berikut bebrapa tipe datanya:
   1. String    = Tipe data String adalah berbentuk sebuah teks, tipe data tersebut bisa kita berikan-
                  pada suatu variable dengan menggunakan tanda petik (""), 
                  baik tanda petik satu('') atau dua("").
                  Contoh: 'nama = "Paijo"'
   2. Number    = Tipe data Number adalah bebentuk sebuah angka, tipe data tersebut bisa kita berikan-
                  pada suatu variabel langsung menggunakan Angka. Bisa kita masukkan angka tersebut-
                  dengan berbagai bentuk angka, misalnya bisa kita masukkan dengan angka berjenis-
                  'Integer', 'Decimal/Float', dan 'Minus'.
                  Contoh: 'angka = 777', 'angkaFloatDecimal = 1.5', 'angkaMinus = -1.5'
   3. Boolean   = Tipe data Boolean adalah hanya berbentuk dua kemungkinan, yaitu True 1(benar)-
                  dan False 0(salah).
                  Contoh: 'boolean = true/false'
   4. Undefined = Tipe data Undefined adalah berbentuk data yang belum diisi datanya.
                  Contoh: 'undefined = NULL'

3. Dalam bahasa pemrograman Javascript, kita bisa mengganti tipe data pada suatu variable seperti-
   contoh ini: 
   Kita bisa melakukan perubahan suatu tipe data 'String' ke tipe data 'Number' dalam Javascript 
   'let dataString = 'Paijo";'
   'console.log(typeof dataString);' = Output nanti 'string'
   Kita bisa langsung ubah tipe datanya seperti ini
   'let dataString = 21;'
   'console.log(typeof dataString);' = Output nanti berubah menjadi 'number'
   Perlu diingat hal semacam ini hanya bisa berlaku di Javascript saja, tidak berlaku pada bahasa-
   pemrograman lainnya seperti 'Java' dan 'CPP/C++' yaa.
